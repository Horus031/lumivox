# Participant Task Script

Please complete the following tasks in Lumivox:

1. Log in to the platform.
2. Create one learning goal.
3. Create one task linked to that goal.
4. Start a focus session and complete it.
5. Open the dashboard and refresh your task risk scan.
6. Generate an AI insight for a task risk card.
7. Generate a weekly reflection.
8. Open Study Rooms, join a room, and send one chat message.
9. Join the voice room briefly and mute/unmute your microphone.
10. Complete the usability questionnaire.
